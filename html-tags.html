The heading tags dictate the size of text that can be used for headings, starting with <h1> as the largest heading, and <h6> as the smallest.

Example:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

The main text of a page goes between the <body> and </body> tags.

Inside the body tags, you can start writing paragraphs using the <p> and </p> tags.

Example: This is a paragraph!

The <ul> </ul> tags create an unordered HTML list. The <ol> </ol> tags created an ordered HTML list. The <li> tag defines a list item.

Example:


Here is an ordered list:
    Milk Sugar Eggs

And here is an unordered list:
    Milk Sugar Eggs
